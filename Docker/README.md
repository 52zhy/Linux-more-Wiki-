===

如何获得帮助

    docker run --rm mysql xsxs

----------

## 你想翻墙
选择有：shadowsocks、ocserv、strongswan、pptpd、nproxy、openvpn、squid

## 你想拥有tomcat集群
得用：nginx、tomcat、mysql、redis

## 你想拥有php集群
得用：nginx、httpd、php、mysql、redis

## 你想拥有zabbix监控
得用：zabbix、php、mysql

## 你想拥有gitlab管理代码
得用：gitlab、mysql、redis

## 你想拥有mysql主从复制
得用：mysql

## 你想拥有不停机、不锁表的mysql主从复制
得用：mysql、systemd

## 你想拥有ssh环境
得用：ssh

## 你想拥有centos7测试环境
得用：systemd
